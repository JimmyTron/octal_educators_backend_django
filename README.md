## Used the following tools

Model Form
Class Based View
Generic Views (for API)
Django Templating Language

## The model has the following attributes

* Input from User

* Title
* Subtitle
* Cover-image(url)
* Status (draft/published)
* Body

* Processed from server-side:

* Article slug
* Author
* Created time(UTC)
* Updated time(UTC)
